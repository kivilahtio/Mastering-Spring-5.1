## Testing

#### Run all tests:

  $ mvn test

#### Run all tests and generate a html report:

  $ mvn surefire-report:report

Access the report with:

  firefox target/site/surefire-report.html

#### Maven dependency tree to figure out dependency issues

  $ mvn dependency:tree
